## 0. 个人情况

我的基本情况是：20岁，体重约61kg，每周4次功能性力量训练，每天至少25分钟 Zone 2 有氧，日常活动量较高，核心目标是维持体重、稳定精力、支持训练表现，同时尽量避免精制碳水、超加工食品和我个人不太信任的高反营养素植物来源。因此，我的饮食框架更偏向高营养密度动物性食物、适量训练碳水和少量低风险植物。


## 1. 宏观结构安排

我的饮食设计哲学满足以下限制：

- **优先保证体重不下降**
- **以动物性食物为基础**
- **碳水主要服务训练，需要多少给多少，不多给**
- **植物只保留低风险、高收益、可消化的部分**

对我个人而言，每天的宏量营养素摄入大概是这样的：

| 蛋白质 | 130–160g |
| 碳水 | 80–150g |

## 3. 饮食优势和权衡考虑

### 3.1 动物性食材为主，减少植物来源反营养素、植酸和凝集素

#### 3.1.1 为什么不吃植物

统营养学往往无条件推崇“多吃植物、全谷物”，却忽略了植物为了防止被吃掉而进化出的化学防御武器（反营养素）。

| **类别** | **规避的高风险食物** | **规避原因及机制** |
| --- | --- | --- |
| **高草酸** | 菠菜、甜菜叶、大量芝麻、各类坚果（如花生、腰果） | 草酸会强效螯合血液中的钙、镁、锌，不仅让高价补充的矿物质白费，还会极大增加年轻人及高蛋白饮食者的泌尿结石风险。 |
| **高凝集素** | 豆类（尤其是未熟透的红腰豆）、花生、夜影科植物（如部分人体质排斥西红柿、茄子） | 凝集素难以被消化，会直接磨损肠道黏膜屏障，引发“肠漏”和隐性系统性炎症，这也是许多人长期饭后胀气、脑雾的元凶。 |
| **高植酸/抗原** | 糙米、全麦面食、燕麦、粗杂粮 | 植物的防御毒素几乎全集中在外壳（麸皮）。植酸会抑制消化酶，导致严重消化不良。这也是本框架坚决弃用“全谷物”，转而选择“白米饭”的核心原因（物理去皮即去毒）。 |
| 植物雌激素 | 大豆 | 干扰内分泌 |
| **生物碱** | 茄子/西红柿 | 组胺/关节炎症 |

#### 3.1.2 那些植物可以吃

| **食物** | **建议量** | **保留理由** |
| --- | --- | --- |
| **牛油果** | 150–200g | 提供单不饱和脂肪酸（抗炎）、极高的天然钾元素。无植酸压力，完美便携的优质脂肪来源。 |
| **熟西兰花** | 150–250g | 提供萝ト硫素（强力激活肝脏二相解毒酶）。**前提：必须彻底煮熟软烂**，以破坏可能干扰甲状腺的物质和粗硬纤维。 |
| **白米饭 / 土豆 / 根茎** | 依训练量浮动 | 这是极其纯净的“幽灵碳水”。根茎类长在地下，缺乏表面毒素；白米饭去壳脱毒。若煮熟放凉，更能产生抗性淀粉，可以平稳血糖并喂养大肠有益菌。 |
| **香蕉** | 训练前后1–2根 | 天然钾，速效碳水。无需经过复杂消化，快速穿透肌肉细胞，为高强度训练提供极其干净的 ATP 爆发力。 |
| **莓果（拒绝日本产）** | 100-150g | 浆果中的王者。提供花青素与微量果糖。果糖可无视胰岛素直接补充肝糖原，不引起全身血糖波动。 |
| **洋葱 / 蒜** | 适量配菜 | 提供有机硫化物，天然杀菌、抗氧化，增强肉类风味。 |
| **绿茶 / 黑咖啡** | 随早餐喝 | 咖啡因结合晨起低胰岛素状态，能强力激活脂肪酸游离（促燃脂）和细胞自噬，提供专注力。 |

### 3.2 低碳饮食框架，非严格生酮，保留优质碳水，兼顾代谢灵活性

#### 3.2.1 为什么要做低碳？

为了对抗“血糖过山车”**与**“胰岛素抵抗”。

通过将碳水控制在每天 100g左右，并将大部分热量交给优质肉类和天然脂肪，胰岛素全天大部分时间处于静息态。这带来了绝对稳定的情绪、清零的餐后困倦、极低的内脏脂肪囤积，以及毫无脑雾的高效专注感。

核心宗旨是：我把自己当作一个糖尿病人对待，我不忍心我的胰岛素和肝脏每天被折腾。

#### 3.2.2 为什么不做严格生酮？

这是踩过深坑后的战略回调。对于一个规律力量训练者，长期摄入 <20g 的极低碳水会引发三大生理灾难：

1. **下丘脑-皮质醇轴（HPA）暴走：** 肝糖原一旦长期见底，大脑在夜间（高耗能 REM 睡眠期）会面临能量真空，强行召唤皮质醇拆解肌肉造糖，导致每天凌晨 4:30 准时带着心悸惊醒。
2. **低 T3 综合征（代谢降速）：** 身体误以为处于“极度饥荒”，主动降低甲状腺活性（FT3 暴跌），导致手脚冰凉、运动掉力量、恢复极慢。
3. **“高蛋白退酮”悖论：** 为弥补碳水热量而大量进食蛋白质，同样会触发胰岛素波动并锁死产酮开关，陷入“糖酮双低”的代谢死局。

#### 3.2.3 代谢灵活性的重要性

给糖烧糖，给油烧油，互不干扰

- **白昼防弹态：** 早餐 0 碳水（靠鸡蛋、牛油果的高脂肪），让大脑完全浸泡在平稳的游离脂肪酸中，高效燃脂。
- **夜间与练后安神态：** 将 100g 左右的干净碳水（白米饭/红薯）作为战略武器“后置”在傍晚。利用训练后肌肉的空窗期（GLUT4 通道大开）吸收糖原防发胖；同时依靠碳水引发的微量胰岛素来镇压皮质醇、合成血清素，稳定夜间睡眠。

### 3.3 进食窗口固定+必要补丁补充

#### 3.3.1 一日两餐的好处

1. **重启肠道保洁系统：** 将进食浓缩在 8 个小时内，每天给肠道留下 16 个小时的绝对空腹期。这能强效激活肠道的**移行性复合运动（MMC）**，自动扫除发酵残渣和多余的有害菌，彻底治愈长期困扰的消化不良、胀气和炎症。
2. **极简时间管理：** 节省中午吃饭时间、避开排队和餐后困倦，可利用中午时间进行学习或运动。

#### 3.3.2 必要补丁的合理性

- **训练补丁：** 无氧力量训练无法直接靠燃烧脂肪供能。练前的一根快碳，可有效保障无氧运动表现
- **睡前补丁：** 当最后一餐距离睡觉超过了 6-8 小时，如果出现微弱的饥饿与心慌，放弃“不能破断食”的无用教条。蜂蜜可以阻断半夜低血糖，果糖可以绕过胰岛素直接补充肝糖原，**在生命系统中，保护深度睡眠的优先级永远大于一切。**

### 3.4 利弊权衡和自我监控

这套饮食的优势是营养密度高、蛋白质量高、血糖波动小、饱腹感强、训练前后碳水更可控；代价是食材成本较高、外食兼容性差、植物多样性较低、对血脂和电解质管理要求更高。因此，它适合被视为一个需要监测和迭代的个人方案，而不是一次性完成的标准答案。

通过以下指标，可以进行系统的量化自我监控：

| 维度 | 指标 |
| --- | --- |
| 体重 | 7日晨起空腹体重平均值 |
| 训练 | 力量表现、心肺状态、恢复速度 |

## 4. 潜在风险

### 4.1 以下指标需要定期（3-6个月）进行测量

| **ApoB** (载脂蛋白B) | 传统低密度脂蛋白 (LDL-C) 在低碳人群中必定会因为“运脂需求增大”而出现生理性升高 (LMHR效应)。重点看 ApoB 和小而密低密度脂蛋白。ApoB 越低，证明动脉粥样硬化的风险越趋近于零。 |
| --- | --- |
| **TG / HDL-C 比值** | 心血管健康程度的金标准。极度干净的饮食会让甘油三酯 (TG) 跌破下限，高密度脂蛋白 (HDL) 大幅飙高。<0.87为最佳。 |
| **hs-CRP** (超敏C反应蛋白) | 衡量体内是否有隐性“系统级炎症”。如果 < 1.0，证明饮食路线完全适配。 |
| **空腹胰岛素 + 血糖** | 用于监控“代谢灵活性”是否还在正轨，计算 HOMA-IR（胰岛素抵抗指数）。夜间加适量慢碳并不会让你得糖尿病，这两个数值会在极低的正常水平保持一条死直线。 |
| **FT3 / TSH** (甲状腺核心项) | 生酮或断食过头，大脑为了节能就会拉低 FT3 (产生低T3综合征)。重点盯紧 FT3 是否回归中位数。 |
| **血清尿酸** (Uric Acid) | 极低碳初期，酮体排出和尿酸排出会竞争同一通道。过了适应期且不再摄入超量蛋白质后，多喝水，尿酸应稳步下降。 |
| **肌酐 / eGFR** | 常年力量训练 + 服用 肌酸补剂，血肌酐往往呈现“假性超标”。看真实的肾脏处理能力，一定要看 eGFR（肾小球过滤率），>90 即可高枕无忧。 |
| **铁蛋白** (Ferritin) | 铁调素机制存在，一般无需担心，定期复查。 |
